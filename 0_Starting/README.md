# Python for Data Science - 0: Starting

This repository contains my solutions for the **42 Python for Data Science - Module 0 (Starting)**. The project introduces Python fundamentals, covering data types, functions, command-line arguments, text processing, functional programming, progress bars, and package creation.

## Exercises

| Exercise | Description |
|----------|-------------|
| **ex00** | Modify basic Python data structures (list, tuple, set, dictionary) and understand mutability. |
| **ex01** | Format and display timestamps using Python's date and time libraries. |
| **ex02** | Create a function that identifies object types and demonstrates Python type checking. |
| **ex03** | Detect and display different "null" values (`None`, `NaN`, `False`, empty string, etc.). |
| **ex04** | Build a CLI program that determines whether a given integer is odd or even with proper error handling. |
| **ex05** | Analyze a text by counting uppercase, lowercase, punctuation, digits, and spaces. |
| **ex06** | Reimplement Python's `filter()` using list comprehensions and build a word filtering program. |
| **ex07** | Encode text into Morse code using dictionaries and command-line arguments. |
| **ex08** | Recreate a simplified version of the `tqdm` progress bar using generators (`yield`). |
| **ex09** | Create and package a reusable Python module installable via `pip`. |
