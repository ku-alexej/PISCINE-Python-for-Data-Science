# Python for Data Science - 1: Array

This repository contains my solutions for the 42 Python for Data Science - Module 1 (Array). The project introduces NumPy arrays and image manipulation, covering array operations, slicing, image loading, cropping, transformations, and basic image filtering while reinforcing clean, modular Python programming.

## Exercises
| Exercise | Description |
|----------|-------------|
| ex00 | Calculate BMI values from height and weight lists and determine whether each BMI exceeds a given limit. |
| ex01 | Slice a 2D array using Python slicing, displaying the original and resulting array shapes. |
| ex02 | Load JPG/JPEG images, validate the input, and display the image dimensions and RGB pixel data. |
| ex03 | Crop, zoom, and display a grayscale section of an image while preserving image information. |
| ex04 | Crop and manually transpose an image without using built-in transpose functions. |
| ex05 | Apply basic image filters (invert, red, green, blue, and grayscale) while maintaining the original image shape. |
