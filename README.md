# Piscine Python

## Description

Similar to a coding bootcamp, this piscine is divided into multiple small projects, with each module focusing on a specific concept, programming paradigm, or technology related to Python.

The goal of this project is to build a strong foundation in Python by progressing step-by-step through essential topics such as basic syntax, data structures, data manipulation, object-oriented programming, and software design concepts.

## Project Structure

The piscine is divided into five modules:

### Python - 0 - Starting
Introduction to Python basics, including syntax, variables, input/output, and fundamental programming concepts.

### Python - 1 - Array
Exploration of arrays, lists, and data manipulation techniques in Python.

### Python - 2 - DataTable
Working with structured data and learning how to handle tables and datasets efficiently.

### Python - 3 - OOP
Introduction to Object-Oriented Programming, including classes, objects, inheritance, and encapsulation.

### Python - 4 - Dod
Application of Python concepts through more advanced design and development practices.
